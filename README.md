# ha-addon-163-gateway
Home Assistant到网易IoT平台的网关插件，用于将HA中的传感器数据同步到网易IoT平台

## 功能
- 自动发现HA中的传感器设备
- 支持温度、湿度、电池等多种属性同步
- 支持智能开关、智能插座、智能断路器的开关状态、电压、电流等多种属性同步
- 定时推送数据到网易IoT平台
- 支持多设备管理和配置

## 安装方法
1. 在Home Assistant中添加仓库：https://github.com/Dmxmj/ha_to_163
2. 在Add-on商店中搜索"HA to 163 Gateway"并安装
3. 配置网关和子设备信息
4. 启动插件
